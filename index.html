Front-end developer?
É uma pessoa responsável pelo desenvolvimento de aplicações web. Essas aplicações são projetadas para funcionar no computador de um usuário final, através de um navegador web. Essas aplicações web são desenvolvidas por meio de linguagens, essas linguagens são interpretadas por navegadores, os famosos (Chrome, Firefox, Safari…) para apresentar uma página da web.
